# Location Insights

An interactive mapping application built with React and React Leaflet for exploring and visualizing geographic data.

## Features

* Interactive maps with zoom and pan support.
* Marker and popup integration.
* Geolocation support.
* Custom map layers and tiles.
* Responsive design for desktop and mobile devices.
* Easy integration with external APIs and location-based services.

## Tech Stack

* React
* React Leaflet
* Leaflet
* JavaScript
* CSS
